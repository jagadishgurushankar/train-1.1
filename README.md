# train-1.1
An Artificial Neural Network (ANN) can predict employee tenure by learning complex, non-linear relationships between demographic, job, and compensation features.
# Dataset
This is a structured HR (Human Resources) employee master dataset where each row represents one employee and columns capture personal, job-related, and employment lifecycle information.
- Demographics & Location: EmployeeID, name, gender, DOB, education, address, city, state, country
- Job & Organization Data: Job_title, Job_Profile, level (e.g., CSuite, Manager), Salary, Active Status
- Employment Metrics: Tenure (years with the company), Notes (change history), Active Status (1 = active, 0 = inactive)
